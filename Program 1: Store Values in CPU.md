# Storing Hex Value in the Memory.
This program loads A and adds B, which will then store in C.
    #### Program
| Position | Memory | Instruction |
|--------|--------|--------|
| 000 |  | Load A |
| 001 |  | Add B |
| 010 |  | Store C |
| 011 |  | 00000 |
| 100 | A | 00111 |
| 101 | B | 00110 |
| 110 | C | 00000 |
| 111 | D | 00000 |
